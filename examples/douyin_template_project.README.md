# douyin_template_project

输入：
- 一个主视频素材

说明：
- 这是模板初始化后的二次编辑示例，请先把 `./assets/input.mp4` 换成你的素材路径。

输出：
- `./output/video/douyin_template_render.mp4`
- `./output/cover/douyin_template_render_cover.jpg`
- `./output/report/douyin_template_render.render.json`
- `./output/report/douyin_template_render.publish.txt`

执行命令：

```bash
./clawcut render -project ./examples/douyin_template_project.json
```

适合：
- 先套模板，再二次微调的项目
